# Supply-Chain-Delivery-Analysis
## Business Context

In supply chain operations, delivery performance is a key factor affecting customer satisfaction and operational efficiency. Delays in shipments can result from multiple factors such as warehouse processing time, transportation issues, or regional operational challenges.

Operations teams often monitor delivery performance to identify bottlenecks and improve service reliability.

## Problem Statement

The objective of this project is to analyze delivery performance data to understand where delays occur and identify potential operational improvement opportunities.

## Data & Tools

This project uses a simulated supply chain dataset containing shipment information such as:

- Shipment date
- Delivery date
- Region
- Delay status
- Delay reason
- Shipment volume

Tools used:
- Excel
- Power BI
- Basic data analysis

## Key Analysis Questions

The analysis focuses on the following questions:

1. Which regions have the highest delivery delay rates?
2. Are delays increasing over time?
3. What are the most common reasons for shipment delays?
4. Are certain shipment volumes associated with higher delays?

## Expected Outcome

The goal of this analysis is to identify patterns in delivery performance and highlight potential operational improvements that could help reduce delays and improve service reliability.
